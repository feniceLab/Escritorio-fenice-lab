## 🚀 Quick Start

### 1. Instale as dependências

```bash
npm install @starken/design-system
```

### 2. Importe o CSS

```jsx
import '@starken/design-system/dist/starken-design-system.css';
```

### 3. Use os componentes

```jsx
<button className="starken-btn starken-btn-primary">
  Começar Agora
</button>
```

## 🎨 Pilares do Design

### Cyber-SaaS
Estética moderna e tecnológica que transmite inovação.

### Premium Dark
Fundos escuros com acentos neon verde que destacam elementos importantes.

### Alta Performance
Otimizado para velocidade e experiência fluida.

### Acessível
Todas as combinações de cores atendem WCAG 2.1 nível AA.

### Consistente
Padrões visuais e funcionais em todos os pontos de contato.
